# Airbnb Clone (Node.js + Express + EJS)

A work-in-progress Airbnb-style web app for managing property listings and reviews.

---

## 🚀 Features
- Create, view, edit, and delete listings
- User-friendly UI with EJS templates
- MongoDB database integration (coming soon)
- Review system (in progress)

---

## 🛠️ Tech Stack
- Node.js
- Express.js
- EJS
- MongoDB & Mongoose (planned)
- Bootstrap/Tailwind (if used)

---
## Setup
1. Clone the repo:
   git clone https://github.com/niharika198/airbnb-clone.git

2. Go into the project folder:
   cd airbnb-clone

3. Install dependencies:
   npm install

4. Start the server:
   node app.js

5. Open your browser at:
   http://localhost:3000
